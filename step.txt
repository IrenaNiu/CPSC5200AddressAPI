1. Activate PostgreSQL server:
    $ cd Library/PostgreSQL/12/bin
    $ psql -U postgres
      Password for user postgres: 123456

2. Create 'my_flask' database:
    postgres=#  CREATE DATABASE my_flask;
    postgres=#  \l 
    postgres=#  \c my_flask

3. Create 'users' table on 'my_flask' database:
    my_flask=#  CREATE TABLE users(
                 id SERIAL PRIMARY KEY,
                 name VARCHAR(255),
                 age VARCAR(255)
                 );
    my_flask=#  \d

4. run server
    $ python app.py
